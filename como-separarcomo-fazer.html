Como fazer a separação?

Separar o lixo corretamente é fundamental para o sucesso da coleta seletiva. Veja como é simples: Tenha duas lixeiras principais em casa: uma para o lixo orgânico/rejeito (restos de comida, papel higiênico, fraldas) e outra para o lixo reciclável (plásticos, papéis, vidros, metais).

Lave as embalagens recicláveis (como potes de iogurte, latas de molho) para remover restos de alimentos antes de descartar. Seque-as se possível. Desmonte caixas de papelão para economizar espaço. Tenha cuidado ao descartar vidros quebrados, embrulhando-os em jornal ou colocando-os dentro de uma caixa para evitar acidentes com os coletores.
